# Deep Learning Project

Step 0 initializes the project structure for the afternoon assignment.

The project will progressively add three independent Keras pipelines:

- Regression
- Binary classification
- Multiclass classification

Later phases will add TensorBoard diagnostics, L2 regularization, early stopping,
Keras Tuner, BatchNorm, logs, and the final Phase 9 metrics comparison.
